# 3DS太鼓达人 不可思议冒险 系统文本工具
## TaikoTextForm
> #### 图形化文本编辑工具
>> #### '保存' 和 '另存为' 两者实现不同, '保存' 功能是将译文覆盖原文, 所以译文长度不能超过原文
>> #### '另存为' 则是根据文件格式重新创建一个文件,理论上译文长度可以无限长
>> #### 请根据实际情况选择使用
## TaikoTextConsole
> #### 命令行版文本工具
> #### 批量导出/生成 文本
	用法 TaikoTextConsole <路径>
> #### 或直接拖拽 dat 或 xlsx 文件到程序上面 会根据文件扩展名进行相应转换
